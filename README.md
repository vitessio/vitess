# Vitess 

Vitess is a cloud-native horizontally-scalable distributed database system built around MySQL.
It allows application code and database queries to remain agnostic to the distribution of data onto multiple database servers.

## Community

Vitess has a growing community. For more information, visit [vitess.io](https://vitess.io/community/).

## Reporting Issues

Report issues on [GitHub](https://github.com/vitessio/vitess/issues). Join our [Slack workspace](https://vitess.io/slack) for live discussions.

## Security

Report security vulnerabilities to [vitess-maintainers](mailto:cncf-vitess-maintainers@lists.cncf.io).

See the [Security](SECURITY.md) page for more details.
